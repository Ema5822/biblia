
# Biblia Web para Netlify

Proyecto estático listo para subir a Netlify.

## Qué incluye
- Lectura por versión, libro y capítulo
- Ir a referencia rápida (ej. Juan 3:16)
- Favoritos guardados en el navegador
- Notas por capítulo guardadas en el navegador
- Tema oscuro/claro
- Tamaño de letra ajustable
- Navegación anterior / siguiente
- Capítulo aleatorio

## Cómo subirlo a Netlify
1. Descomprimí el ZIP.
2. Entrá a Netlify.
3. Arrastrá la carpeta o el archivo `index.html` al panel de deploy.
4. Listo.

## Importante
Esta web usa una API pública para obtener capítulos bíblicos desde el navegador:
- Free Use Bible API: https://bible.helloao.org/
La API ofrece traducciones, libros y capítulos vía JSON sin clave de API.

## Archivos
- `index.html`: aplicación completa en un solo archivo
